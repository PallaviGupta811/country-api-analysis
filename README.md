
# Country Data Analysis using REST API

## Overview

This project fetches live country data from the REST Countries API and performs basic analysis using Python and Pandas.

## Features

* Fetch data using API
* Convert JSON to DataFrame
* Analyze population and area
* Identify largest country and most populated region

## Tools Used

* Python
* Pandas
* Requests
* Google Colab
